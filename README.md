# Legendary UNO Arena

نسخة Flask + Socket.IO للعبة UNO داخل غرف مباشرة مع صوت WebRTC وواجهة عربية.

## الموجود داخل المشروع
- لعبة UNO كاملة بمنطق الجولة والأوراق الخاصة والتحدي على +4 ونداء UNO.
- 108 بطاقة SVG فعلية داخل `static/cards` + بطاقة ظهر.
- لوبي داخل الغرفة مع ملف شخصي ورفع صورة اللاعب.
- انتقال المضيف تلقائيًا إلى اللاعب التالي في ترتيب الدخول عند خروج المضيف.
- حد أدنى لاعبين قبل بدء الجولة.
- إعدادات الجولة للمضيف فقط.
- صوت مباشر دائم مع زر كتم المايك وزر كتم السماعات.
- شخصيات مختارة بصور ويب مباشرة مستوحاة من Resident Evil.
- ملفات جاهزة للنشر على Railway.

## التشغيل المحلي
```bash
pip install -r requirements.txt
python server.py
```

## Railway
- ارفع المشروع إلى GitHub.
- اربطه مع Railway.
- اترك `Procfile` كما هو.
- أضف متغير بيئة واحد على الأقل:
  - `SECRET_KEY=any-strong-value`
